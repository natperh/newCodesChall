# Flujo de Negocio
```mermaid
graph TD
    A[Usuario introduce primer número] --> B[Programa lee primer número y lo almacena en NUM1]
    B --> C[Usuario introduce segundo número]
    C --> D[Programa lee segundo número y lo almacena en NUM2]
    D --> E[Programa calcula suma de NUM1 y NUM2 y la almacena en RESULTADO]
    E --> F[Programa muestra resultado de la suma al usuario]


Este diagrama de flujo muestra los pasos que sigue el programa SUMA para calcular la suma de dos números enteros introducidos por el usuario. Cada paso se representa con un nodo (A, B, C, D, E y F) y las flechas indican el flujo de datos entre ellos.

Espero que este diagrama te sea útil para visualizar el flujo de datos del programa SUMA. ¡Si tienes alguna pregunta o necesitas más información, no dudes en preguntar!
```
